西南科技大学蓝鲸Sass开发课程

课程介绍

本课程由学校与腾讯蓝鲸联合打造，旨在帮助学生全面了解并掌握 SaaS 开发的核心技能，以及蓝鲸平台的应用与实践。课程在原有基础上增设了 SaaS 开发基础知识前置模块，帮助大家夯实基础，顺利衔接后续内容。

核心课程

蓝鲸应用开发

流程 SaaS 开发全流程 · 从初始化到部署上线
平台 蓝鲸 CMDB · 作业平台 · BKVision
实战 Vue 3 + 蓝鲸框架 · 开发"游戏主机检测系统"

通过以上模块的学习，你将建立起扎实的 SaaS 开发基础，为深入蓝鲸平台开发做好充分准备。

课程安排

Python基础
  • 第零课：课程环境搭建（Windows）

Django基础
  • 第三节课：MTV架构、路由视图与ORM实操
  • 第四节课：认证鉴权、中间件与REST API
蓝鲸项目实战
  • 第五节课：云原生开发平台与SaaS应用部署

  • 第七节课：《蓝鲸作业平台》理论与实验
  • 第八节课：《蓝鲸BKVision实战开发》理论与实验
