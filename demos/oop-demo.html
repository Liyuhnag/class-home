Python 面向对象 — 交互演示

本页面专为有 C 语言基础的同学设计，通过对比和可视化帮助理解 Python 面向对象编程。封装、继承、多态、魔术方法一网打尽。

1. 面向对象：从 C 结构体到 Python 类

💡 你已经知道的：C 语言的结构体 C 的 struct 可以把相关数据打包在一起，但操作这些数据的函数是独立的。
Python 的 class 把数据（属性）和操作它的函数（方法）打包在一起，更内聚。

C 语言：数据与函数分离

typedef struct {
    char name[50];
    int age;
    float score;
} Student;

// 函数在结构体外面
void introduce(Student *s) {
    printf("我是%s，%d岁\n",
           s->name, s->age);
}

int is_passed(Student *s) {
    return s->score >= 60;
}

// 使用
Student s = {"Alice", 20, 90};
introduce(&s);

Python：数据与方法在一起

class Student:
    def __init__(self, name,
                 age, score=0):
        self.name = name
        self.age = age
        self.score = score

    # 方法在类内部
    def introduce(self):
        print(f"我是{self.name}，"
              f"{self.age}岁")

    def is_passed(self):
        return self.score >= 60

# 使用
s = Student("Alice", 20, 90)
s.introduce()

2.1 封装：保护内部数据

💡 类比：ATM 机 你不能直接修改银行数据库里的余额（相当于 _balance 私有），只能通过存款/取款接口操作（相当于 deposit() / withdraw()）。
📋BankAccount 封装代码
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self._balance = balance  # 私有属性（约定）

    def deposit(self, amount):
        """存款接口"""
        if amount <= 0:
            raise ValueError("存款金额必须大于 0")
        self._balance += amount
        return self._balance

    def withdraw(self, amount):
        """取款接口"""
        if amount > self._balance:
            raise ValueError("余额不足")
        self._balance -= amount
        return self._balance

    @property
    def balance(self):
        """只读属性"""
        return self._balance

# 使用
account = BankAccount("Alice", 1000)
account.deposit(500)      # ✅ 通过接口
account._balance = -999   # ⚠️ 可以直接改，但不应该

2.2 继承：代码复用

💡 类比：公司职级 Employee（员工）是基类，有 name / salary / work()。
Manager（经理）继承 Employee，额外有 manage() 方法。
Engineer（工程师）继承 Employee，额外有 code() 方法。
📋继承代码
class Employee:
    def __init__(self, name, salary):
        self.name = name
        self.salary = salary

    def work(self):
        return f"{self.name} 在工作"

class Manager(Employee):
    def manage(self):
        return f"{self.name} 在管理团队"

class Engineer(Employee):
    def code(self):
        return f"{self.name} 在写代码"

# 使用
m = Manager("Bob", 15000)
e = Engineer("Carol", 12000)
print(m.work())      # ✅ 继承自 Employee
print(m.manage())    # ✅ Manager 特有
print(e.code())      # ✅ Engineer 特有
🌳继承关系可视化
🎯 关键点：子类自动拥有父类的所有属性和方法。super().__init__() 相当于「先初始化父类部分」，就像 C++ 的成员初始化列表调用基类构造函数。

2.3 多态：同一接口，不同行为

💡 类比：遥控器 同样按「播放」按钮，接的是电视就播电视，接的是音响就播音乐。
多态让你写 animal.speak()，不管 animal 是 Dog 还是 Cat，都会调用各自的叫声。
📋多态代码
class Animal:
    def speak(self):
        raise NotImplementedError

class Dog(Animal):
    def speak(self):
        return "汪汪！"

class Cat(Animal):
    def speak(self):
        return "喵喵～"

class Bird(Animal):
    def speak(self):
        return "叽叽喳喳"

# 多态：同一接口，不同行为
animals = [Dog(), Cat(), Bird()]
for animal in animals:
    print(animal.speak())  # 自动调用各自的 speak()

2.4 魔术方法：让对象支持运算符

💡 类比：重载运算符 学过 C++ 的同学知道「运算符重载」。Python 的魔术方法是同样的思想：
定义 __add__ 就支持 +，定义 __str__ 就支持 print()。
📋Vector 魔术方法代码
import math

class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other):    # v1 + v2
        return Vector(self.x + other.x, self.y + other.y)

    def __eq__(self, other):     # v1 == v2
        return self.x == other.x and self.y == other.y

    def __abs__(self):           # abs(v)
        return math.sqrt(self.x**2 + self.y**2)

    def __str__(self):           # print(v)
        return f"Vector({self.x}, {self.y})"

v1 = Vector(1, 2)
v2 = Vector(3, 4)
print(v1 + v2)    # Vector(4, 6)
print(v1 == v2)   # False
print(abs(v2))    # 5.0
print(v1)         # Vector(1, 2)

2.5 三种方法：实例 / 类 / 静态

类型装饰器第一个参数可访问用途
实例方法无self（实例）实例数据 + 类数据普通业务逻辑
类方法@classmethodcls（类）类数据工厂方法、替代构造器
静态方法@staticmethod无都不能直接访问纯工具函数，逻辑上属于类
📋三种方法代码
class Student:
    school = "西南科技大学"   # 类属性

    def __init__(self, name, score):
        self.name = name           # 实例属性
        self.score = score

    def greet(self):               # 实例方法
        return f"我是{self.name}，来自{self.school}"

    @classmethod
    def from_string(cls, s):       # 类方法（工厂）
        name, score = s.split(',')
        return cls(name, int(score))

    @staticmethod
    def is_passed(score):          # 静态方法
        return score >= 60

# 使用
s1 = Student("Alice", 90)
s2 = Student.from_string("Bob,75")  # 工厂方法
print(Student.is_passed(55))        # False

3. 代码参考

完整的 OOP 示例：学生管理

综合运用封装、继承、多态
class Person:
    def __init__(self, name, age):
        self.name = name
        self._age = age  # 封装

    @property
    def age(self):
        return self._age

    def __str__(self):
        return f"{self.name}({self._age}岁)"

class Student(Person):  # 继承
    def __init__(self, name, age, student_id):
        super().__init__(name, age)  # 调用父类构造
        self.student_id = student_id
        self._scores = []

    def add_score(self, subject, score):
        self._scores.append((subject, score))

    @property
    def gpa(self):
        if not self._scores:
            return 0
        return sum(s for _, s in self._scores) / len(self._scores)

    def report(self):  # 多态：子类有自己的实现
        return f"学生 {self} - 平均分: {self.gpa:.1f}"

class GradStudent(Student):
    def __init__(self, name, age, student_id, advisor):
        super().__init__(name, age, student_id)
        self.advisor = advisor

    def report(self):  # 覆盖父类方法
        base = super().report()
        return f"{base} - 导师: {self.advisor}"

# 多态使用
people = [
    Student("Alice", 20, "S001"),
    GradStudent("Bob", 24, "G001", "Dr. Wang")
]
people[0].add_score("数学", 90)
people[0].add_score("英语", 85)

for p in people:  # 同一接口，不同行为
    print(p.report())
