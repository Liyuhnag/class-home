底层原理：CPU 在干嘛？

你可能会问：程序不是一直在运行吗？怎么能「同时」做三件事？关键在于——大部分时间 CPU 其实在「干等」。

🧠 CPU 很快，网络很慢
CPU 每秒能执行几十亿条指令，但一个网络请求往往需要几十到几百毫秒才能返回。这期间 CPU 什么计算都不用做，只是在「等数据回来」。
这种「发出请求 → 等待响应」的过程叫 I/O 操作（Input/Output，输入输出），包括：网络请求、读写文件、查数据库等。

核心发现：同步模式下，CPU 发完请求后就「卡住」了，必须等数据回来才能做下一件事。异步模式下，CPU 发完 A 的请求后不等，而是立刻去发 B、C 的请求。等所有数据回来，再集中处理。CPU 干的活一样多，但总时间大幅缩短。

总结「让出 CPU」的过程：
① 任务 A 执行到 await（发送网络请求），主动把 CPU 让出来；
② 事件循环把 CPU 交给任务 B，B 也执行到 await，又让出；
③ CPU 交给任务 C，C 也让出；
④ 三个请求都在网络上飞，CPU 暂时空闲；
⑤ 哪个请求先回来，CPU 就先去处理谁。

这就是为什么 Python 的 asyncio 虽然是单线程，却能高效处理大量 I/O 任务——它不是真的「同时做」，而是在「等待间隙」快速切换。

常见错误：异步里用了阻塞操作

很多初学者在异步函数里不小心用了 time.sleep()（阻塞操作），结果异步完全失效。点击运行看看区别。

❌ 错误写法：time.sleep（阻塞）

    time.sleep(1)  # 卡住整个程序！

记住这条规则：在 async def 函数里，所有「等待」操作都必须用 await 版本。time.sleep → await asyncio.sleep，requests.get → await aiohttp。否则异步就白写了。

四个关键词

下面是 Python 异步编程中最核心的四个概念。别被英文名吓到，看完比喻就好懂了。

📋
协程 (Coroutine)
用 async def 定义的函数。和普通函数不同，它可以在等待时「暂停」，让别的任务先做。
💡 比喻：你在等外卖时先去写作业，外卖到了再去拿——你就是一个「协程」。
⏸️
await
写在异步函数里，表示「这里要等一下」。等待期间程序不会卡住，而是去做别的事。
💡 比喻：就是「等外卖」的那个动作。你说了 await，程序就知道这里有空档。
🔄
事件循环 (Event Loop)
负责调度所有协程的「总管」。它不断检查：谁等完了？轮到谁继续？
💡 比喻：像食堂阿姨，看哪个窗口空了就叫下一个人上来。
📦
asyncio
Python 自带的异步工具库。提供了事件循环、sleep、gather 等常用功能，开箱即用。
💡 比喻：就是 Python 给你准备好的「异步工具箱」。

代码长什么样？

理解了概念，再来看代码就很自然了。注释里标了关键步骤，对照上面的动画理解。

同步 vs 异步对比
import asyncio
import time

# ========== 同步版本（一个一个来）==========
def sync_download(name, seconds):
    print(f"  [{name}] 开始下载...")
    time.sleep(seconds)          # 程序卡在这里，什么都干不了
    print(f"  [{name}] 完成")

start = time.time()
sync_download("文件A", 1)       # 先下载 A，等 1 秒
sync_download("文件B", 1)       # A 完了才开始 B，又等 1 秒
sync_download("文件C", 1)       # B 完了才开始 C，再等 1 秒
print(f"同步总耗时: {time.time() - start:.1f}s\n")  # → 3.0s

# ========== 异步版本（同时下载）==========
async def async_download(name, seconds):
    print(f"  [{name}] 开始下载...")
    await asyncio.sleep(seconds) # 等待期间去做别的任务
    print(f"  [{name}] 完成")

async def main():
    start = time.time()
    await asyncio.gather(        # 关键！gather 让三个任务同时开始
        async_download("文件A", 1),
        async_download("文件B", 1),
        async_download("文件C", 1),
    )
    print(f"异步总耗时: {time.time() - start:.1f}s")  # → 1.0s

await main()
