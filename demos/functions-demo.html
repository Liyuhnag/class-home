Python 函数 — 交互演示

本页面专为有 C 语言基础的同学设计，通过对比和可视化帮助理解 Python 函数的各种特性。

1. 从 C 函数到 Python 函数

🎯 核心区别C 的函数像「严格的工厂流水线」：必须声明所有类型，只能返回一个值。Python 的函数像「灵活的工作台」：不需要类型声明，可以返回多个值，函数本身是对象。
特性C 语言Python
返回类型必须声明 int func()不需要（可选 -> int）
参数类型必须声明不需要（可选注解）
默认参数❌ C99 不支持✅ def f(a, b=10)
可变参数va_list 复杂易错✅ *args, **kwargs
多返回值只能一个，靠指针模拟✅ 直接返回元组
函数作参数函数指针（复杂）✅ 一等公民，直接传
嵌套函数/闭包❌ 不支持✅ 支持
匿名函数❌✅ lambda x: x*2

4. 默认参数陷阱（经典 Bug）

⚠️ 为什么会有这个 Bug？默认参数值在函数定义时就计算好，只算一次。如果是可变对象（list/dict），所有调用共享同一个对象，会互相污染。
🐛 默认参数陷阱演示

❌ 错误写法

def bad_append(item, lst=[]):
    lst.append(item)
    return lst

print(bad_append(1))  # [1]
print(bad_append(2))  # [1,2] ← 共享！
print(bad_append(3))  # [1,2,3] ← 越来越长！

✅ 正确写法

def good_append(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst

print(good_append(1))  # [1]
print(good_append(2))  # [2] ← 各自独立

6. 闭包：函数记住了外层变量

💡 类比：带私有状态的函数对象闭包 = 内层函数 + 它捕获的外层变量。C 实现同效果需结构体+函数指针，Python 闭包几行搞定。
三个条件：① 有嵌套函数 ② 内层函数引用外层变量 ③ 外层函数返回内层函数。
🔍 闭包形成步进器
步骤 1/5
🎯 nonlocal：内层函数若要修改外层变量，必须用 nonlocal 声明；若只读取则不需要。这和 C 的自由变量读写语义一致。
🔒 闭包计数器 & make_multiplier 演示
# 闭包 1：计数器（带状态）
def make_counter(start=0):
    count = start  # 捕获的外层变量
    def counter():
        nonlocal count
        count += 1
        return count
    return counter

c1 = make_counter(0)   # 独立状态
c2 = make_counter(10)  # 独立状态

# 闭包 2：乘法器（工厂）
def make_multiplier(n):
    def multiply(x):
        return x * n   # 捕获 n
    return multiply

double = make_multiplier(2)
triple = make_multiplier(3)

⚠️ 闭包经典陷阱：循环中创建函数
# ❌ 陷阱：所有函数共享同一个 i
funcs = []
for i in range(3):
    funcs.append(lambda: i)  # 捕获变量 i
# 结果全是 2（循环结束后 i=2）

# ✅ 修复：用默认参数固定值
funcs = []
for i in range(3):
    funcs.append(lambda x=i: x)  # 值拷贝
# 结果分别是 0, 1, 2

闭包：记忆化装饰器
def memoize(func):
    cache = {}
    def wrapper(*args):
        if args not in cache:
            cache[args] = func(*args)
        return cache[args]
    return wrapper

@memoize
def fib(n):
    if n < 2: return n
    return fib(n-1) + fib(n-2)

print(fib(40))  # 瞬间完成
