Python 异常处理 — 交互演示

异常处理是健壮代码的基础。专为有 C 语言基础的同学设计——C 用返回值表示错误，Python 用异常对象传递错误信息。

1. try-except-else-finally 步进器

💡 类比：C 的错误码 vs Python 的异常 C 用 if (ret < 0) return -1; 逐级传递错误，Python 用 raise 直接跳转到最近的 except，代码更简洁。
