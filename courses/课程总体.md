本课程由学校与腾讯蓝鲸联合打造，旨在帮助学生全面了解并掌握SaaS开发的核心技能，以及蓝鲸平台的应用与实践。为了让同学们更好地完成后续的蓝鲸SaaS开发课程，我们在原课程基础上增设了SaaS开发基础知识前置模块，帮助大家夯实基础，顺利衔接后续内容。


前置课程主要分为两部分：Python 基础与Django 基础。

Python 基础部分将系统学习以下内容：  
• 核心语法：数据类型、输入输出、条件语句、循环语句、推导式  

• 基本数据结构与函数  

• 面向对象编程、装饰器、异常处理、文件操作  

• 同步与异步编程、并发编程  

• HTTP 与网络请求、日志系统、单元测试  

Django 基础部分将涵盖：  
• Django 的 MTV 架构与环境搭建  

• 路由系统、视图、ORM 介绍  

• 模型定义、数据库迁移、CRUD 操作  

• Admin 后台配置、用户认证、Session 与 Cookie  

• 中间件、Restful API 设计、RestFramework  

• 认证与权限、分页、异步任务、定时任务  

通过这两个模块的学习，你将建立起扎实的 SaaS 开发基础，为后续深入蓝鲸平台开发做好充分准备。

在蓝鲸应用开发部分，课程将分为两个阶段展开：

第一阶段：SaaS 开发流程
首先，我们将系统学习基于蓝鲸平台的SaaS应用开发全流程，帮助你掌握从项目初始化、配置、开发、调试到部署上线的完整路径。

第二阶段：实战开发——游戏主机检测系统
接着，我们将进入实战环节，综合运用蓝鲸平台的各项核心服务。你将学习如何集成并使用：
• 蓝鲸CMDB 进行资产管理与建模

• 蓝鲸作业平台（Job） 实现任务调度与脚本执行

• 蓝鲸可视化平台（bkvision） 等进行数据监控与展示

本实战项目将采用 Vue 3 前端框架与蓝鲸开发框架，带领你从零开始，动手开发一个功能完整的“游戏主机检测系统”，真正体验企业级SaaS应用的开发过程。
