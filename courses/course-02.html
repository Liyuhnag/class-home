第二节课

课程目标

  1. 掌握 Python 函数定义、参数机制、作用域与闭包。
  2. 理解面向对象编程：封装、继承、多态与魔术方法。
  3. 学会使用装饰器增强函数功能。
  4. 掌握异常处理机制与自定义异常。
  5. 熟练文件操作与 JSON/CSV 数据处理。
  6. 了解同步与异步编程、并发编程基础。
  7. 掌握 HTTP 请求与 requests 库的使用。
  8. 了解日志系统与单元测试基础。

课程内容

1. 函数

▶ 查看 Python 函数交互演示

🏋️ 练习题 — 函数

  • 入门写一个函数 calc(a, b, op='+')，支持加减乘除四种运算，op 默认为加法。提示：用 if-elif 判断 op 的值，注意除零异常。
  • 入门写一个函数 flatten(*args)，接收任意数量的列表并把它们合并成一个列表返回。提示：用 *args 接收多个列表，用 + 或 extend 合并。
  • 进阶利用闭包实现一个 make_multiplier(n)，返回一个函数，该函数将输入乘以 n。例如 double = make_multiplier(2); double(5) == 10。提示：外层函数接收 n，内层函数捕获 n 形成闭包。
  • 进阶用 map + filter + lambda 完成：给定学生列表 [{'name':'Alice','score':90}, ...]，筛选及格（≥60）的学生，按分数降序，返回姓名列表。提示：filter 筛选 → sorted 排序 → map 取名字，全部用 lambda。
  • 挑战实现一个 memoize 装饰器（用闭包），让斐波那契函数 fib(40) 在毫秒内完成。要求不使用 functools.lru_cache。提示：闭包捕获一个 dict 作为缓存，key 是参数，value 是结果。

2. 面向对象

本节内容包含交互式动画演示，专为有 C 语言基础的同学设计，点击下方按钮前往独立演示页面：

▶ 查看 Python 面向对象交互演示

🏋️ 练习题 — 面向对象

  • 入门创建一个 Rectangle 类，有 width 和 height 属性，实现 area() 和 perimeter() 方法，以及 __str__ 方法。提示：面积 = width × height，周长 = 2×(width+height)。
  • 入门给 Rectangle 添加 @property 保护 width 和 height，确保设值时不能为负数（抛出 ValueError）。提示：用 _width 存储私有值，@property + @width.setter 来验证。
  • 进阶创建 Shape 基类（有抽象方法 area()），让 Rectangle 和 Circle 继承它，实现多态：for s in shapes: print(s.area())。提示：基类 area() 可以 raise NotImplementedError，子类分别实现。
  • 进阶为 Rectangle 实现 __eq__（面积相等则相等）和 __lt__（按面积比较），使其支持排序：sorted(shapes)。提示：__lt__ 返回 self.area() < other.area()。
  • 挑战实现一个 Stack 类，支持 push、pop、peek，并实现 __len__、__contains__、__iter__，使其支持 len(s)、x in s 和 for x in s。提示：内部用列表存储，__iter__ 返回 iter(self._data)。

3. 装饰器

本节内容包含交互式动画演示，专为有 C 语言基础的同学设计，点击下方按钮前往独立演示页面：

▶ 查看 Python 装饰器交互演示

🏋️ 练习题 — 装饰器

  • 入门写一个 log_call 装饰器，在函数执行前后分别打印「开始执行 xxx」和「执行完毕 xxx」。提示：用 @wraps(func) 保留原函数名，通过 func.__name__ 获取名字。
  • 入门写一个 timer 装饰器，打印函数执行耗时（用 time.perf_counter()）。用它装饰一个计算 1~N 之和的函数。提示：在 wrapper 内记录开始时间，函数执行后计算差值。
  • 进阶写一个带参数的 @repeat(n) 装饰器，让被装饰函数重复执行 n 次。提示：需要三层嵌套：repeat(n) → decorator(func) → wrapper(*args, **kwargs)。
  • 进阶写一个 @validate_args 装饰器，检查被装饰函数的所有参数是否不为 None，若有 None 则抛出 ValueError 并指明哪个参数为空。提示：用 func.__code__.co_varnames 获取参数名，zip 对应 args 值逐一检查。
  • 挑战实现一个 @retry(max_retries, exceptions) 带参数装饰器，在指定异常发生时自动重试，超过次数后重新抛出异常，并打印每次重试的次数和错误信息。提示：三层嵌套，exceptions 是异常类型元组，用 for attempt in range(max_retries) 控制重试。

4. 异常处理

本节内容包含交互式动画演示，专为有 C 语言基础的同学设计，点击下方按钮前往独立演示页面：

▶ 查看 Python 异常处理交互演示

🏋️ 练习题 — 异常处理

  • 入门写一个函数 safe_divide(a, b)，捕获除零异常并返回 None，其他异常正常抛出。提示：只捕获 ZeroDivisionError，不要写 except Exception 一把抓。
  • 入门验证 else 和 finally 的执行顺序：写一段代码，分别在有异常和无异常时打印各子句的执行情况。提示：在 try/except/else/finally 各块里加 print，对比两种情况输出。
  • 进阶自定义异常类 InsufficientFundsError(Exception)，在银行账户取款不足时抛出，并附带当前余额和尝试取款金额信息。提示：在 __init__ 里保存额外信息，__str__ 返回有意义的错误信息。
  • 进阶用 try-except 包裹一个读取用户输入的循环，直到用户输入合法整数才退出，非整数输入给出提示并继续。提示：捕获 ValueError，用 while True + break 控制循环。
  • 挑战捕获底层 ValueError 后，用 raise RuntimeError('处理失败') from e 重新抛出，打印 __cause__ 属性验证异常链保留了原始信息。提示：from e 设置 __cause__，可以用 except RuntimeError as err: print(err.__cause__) 查看。

5. 文件操作

本节内容包含交互式动画演示，点击下方按钮前往独立演示页面：

▶ 查看 Python 文件操作交互演示

🏋️ 练习题 — 文件操作

  • 入门用 with open(...) 写一个函数，统计一个文本文件中的行数、单词数、字符数并返回字典。提示：行数用 len(lines)，单词数用 split()，字符数用 len()。
  • 入门将一个学生列表（包含 name/score/grade 字段）写入 CSV 文件，再读取回来还原成字典列表。提示：用 csv.DictWriter 写入，csv.DictReader 读取，注意 newline='' 参数。
  • 进阶实现一个简单的 JSON 配置文件管理器：支持读取、更新单个键值、保存，文件不存在时自动创建默认配置。提示：读取时 try-except FileNotFoundError，保存时用 json.dump(indent=2)。
  • 进阶用 @contextmanager 实现一个 timer(name) 上下文管理器，在 with 块开始和结束时分别打印时间戳，计算并打印代码块耗时。提示：from contextlib import contextmanager，yield 前记录开始时间，yield 后计算差值。
  • 挑战读取一个包含多列数据的 CSV 文件，按某一列分组统计（如按 grade 分组统计平均分），结果写入新的 JSON 文件。提示：用 csv.DictReader 读取，用字典按 grade 分组累加，最后 json.dump 写出。

6. 同步与异步编程

本节内容包含交互式动画演示，点击下方按钮前往独立演示页面：

▶ 查看同步与异步交互演示

🏋️ 练习题 — 同步与异步编程

  • 入门用 asyncio 写一个异步函数 fetch_data(url, delay)，模拟网络请求（用 await asyncio.sleep(delay)），并用 asyncio.run() 运行它。提示：async def 定义协程，await 挂起等待，asyncio.run() 启动事件循环。
  • 入门同时发起 3 个异步任务（延迟分别为 1s、2s、3s），用 asyncio.gather() 并发执行，验证总耗时约为 3s 而不是 6s。提示：asyncio.gather(*tasks) 并发运行所有任务。
  • 进阶用 async for 实现一个异步生成器，每隔 0.5s 产生一个斐波那契数，共产生 10 个。提示：使用 async def + yield 定义异步生成器，async for 消费。

▶ 查看单元测试与 pytest 交互演示

🏋️ 练习题 — 单元测试

  • 入门为 calc(a, b, op) 函数（来自函数练习 1）写 unittest 测试用例，覆盖加减乘除和除零的边界情况。提示：继承 unittest.TestCase，每个测试方法以 test_ 开头，用 assertEqual 和 assertRaises。
